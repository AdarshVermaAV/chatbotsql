Install ODBC Driver 17 for SQL Server

 Steps to Install:
1. Download the driver from Microsoft: [ODBC Driver for SQL Server](https://docs.microsoft.com/en-us/sql/connect/odbc/download-odbc-driver-for-sql-server).
2. Install it based on your operating system (Windows/macOS/Linux).
